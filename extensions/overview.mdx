The `shivammathur/extensions` Homebrew tap provides pre-built bottles for more than 60 PHP extensions spanning PHP 5.6 through PHP 8.6. Every formula is versioned per PHP release, so you install exactly the extension build that matches the PHP version active on your machine. The tap works with PHP from both the `shivammathur/php` tap and `homebrew/core`.

## Quick start

Before installing any extension, make sure the tap is added and the target PHP version is installed:

```zsh
brew update
brew tap shivammathur/extensions
```

Then install the extension you need, substituting the PHP version number:

```zsh
brew install shivammathur/extensions/<extension>@<php-version>
```

<Tip>
  If you use Apache or Nginx with php-fpm, restart the web server after installing or upgrading any extension so the new `.so` file is loaded.
</Tip>
