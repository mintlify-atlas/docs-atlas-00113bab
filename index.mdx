`shivammathur/homebrew-extensions` is a Homebrew tap that provides pre-built formulae for PHP extensions covering PHP 5.6 through 8.6. It works alongside the official `homebrew/core` tap or the `shivammathur/php` tap, letting you install extensions like Xdebug, Redis, Imagick, Swoole, gRPC, and many more with a single `brew install` command.

## Get up and running

<Steps>
  <Step title="Add the tap">
    Register the `shivammathur/extensions` tap with Homebrew:

    ```zsh
    brew tap shivammathur/extensions
    ```
  </Step>
  <Step title="Install an extension">
    Install any extension by specifying its name and PHP version. For example, Xdebug on PHP 8.4:

    ```zsh
    brew install shivammathur/extensions/xdebug@8.4
    ```
  </Step>
  <Step title="Verify the installation">
    Confirm the extension is loaded by PHP:

    ```zsh
    php -m | grep xdebug
    ```
  </Step>
</Steps>
